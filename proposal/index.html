Mesh Morphing & Animation Toolkit

Names: Brandon Lai, Tamnhi Vu, Samuel German, Tara Pande

Link to webpage: blai57.github.io/proposal/index.html

Summary: We will develop an interactive tool for 3D mesh deformation and animation, enabling smooth transitions between shapes using Bézier-based interpolation, keyframe animation, and Loop subdivision. The toolkit will allow users to manipulate meshes in real-time and visualize topology changes, serving as a bridge between geometric modeling and animation pipelines.

Problem Description

3D mesh modeling and animation often require complex, computationally intensive techniques to create smooth, natural deformations. Existing tools either lack intuitive interfaces or produce unrealistic mesh transformations. This project addresses these challenges by providing an accessible, mathematically rigorous approach to mesh morphing that preserves geometric integrity while enabling creative geometric transformations.

The core challenge lies in developing interpolation methods that:

  • Maintain mesh topology
  • Preserve vertex relationships
  • Create visually smooth transitions
  • Provide real-time interactive manipulation

We will solve this by extending geometric modeling techniques learned in our homework, specifically leveraging Bézier curve interpolation, half-edge data structures, and advanced mesh processing algorithms.
